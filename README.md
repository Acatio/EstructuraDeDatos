# Estructuras de Datos en Java

Este repositorio contiene implementaciones básicas de estructuras de datos como parte de mi formación en Ingeniería en Softwaree.

## Contenido

- Listas enlazadas simples y dobles

- Bicolas

- Árboles

- Grafos

Practicar la lógica y comprensión de estructuras fundamentales que se utilizan en programación y desarrollo de software.

- Java
- IDE: NetBeans 
- Consola (sin interfaz gráfica)
